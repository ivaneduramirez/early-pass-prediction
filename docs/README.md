# Documentación del proyecto

Predicción temprana de aprobación por asignatura a partir del primer parcial (UTMACH).

- **[USO_MODELO.md](USO_MODELO.md)** — cómo usar el predictor (CLI y API), interpretación y mantenimiento.
- **[RESULTADOS.md](RESULTADOS.md)** — metodología, validación (backtest de 10 períodos), ablaciones y resultados, con figuras.
- **[PRESENTACION.md](PRESENTACION.md)** — guion de presentación (13 láminas + apéndice).

> Este repositorio publica la **demo web** (raíz) y la **documentación** (esta carpeta).
> El pipeline de entrenamiento (`*.py`), los modelos entrenados (`*.joblib`) y los datos
> crudos **no** se incluyen. Las figuras están en `figuras/`.
